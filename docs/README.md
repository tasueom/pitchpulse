# pitchpulse
A football data platform providing live matches, standings, and stat-based rankings with a focus on real-time flow and clarity.

본 프로젝트는 초기 단계에서 Mock 데이터를 활용해
프론트엔드 화면 흐름을 먼저 확정한 뒤,
동일한 UI 구조를 유지한 채 백엔드를 점진적으로 구현하는 방식을 채택했습니다.

이 과정에서 데모 프론트엔드는
화면 구조와 데이터 흐름을 빠르게 검증하기 위한 용도로
Cursor와 같은 AI 기반 개발 도구를 활용해 구성했습니다.

이를 통해 API 설계, 데이터 책임 분리,
그리고 캐싱 및 저장소 도입 시점에 대한
의사결정을 명확히 드러내는 것을 목표로 했습니다.
